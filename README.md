# Simple Stable

Simple CDP-backed stablecoin

## Testing
Enter the dev shell via
```
nix develop
```
And run the tests:
```
forge test
```

## Real Network Config

When it comes time to deploy, copy [.env-example](.env-example) to `.env` and set the variables.
